# Security Policy

Do not report suspected vulnerabilities in a public issue.

Use GitHub's private vulnerability reporting for this repository when the
problem is specific to the formula or tap. For issues in the installed agent,
service setup, privilege boundary, or metadata endpoint, follow the private
reporting instructions in the
[aws-metadata-agent security policy](https://github.com/so1omon563/aws-metadata-agent/security/policy).

Include the affected formula version, Homebrew version, macOS version and
architecture, reproduction steps, and impact. Remove credentials, account IDs,
profile names, identities, and organization-specific information.
